# Front-end for HuaynoData.com

## How to use
- git clone https://github.com/amigocloud/amigo-peru-rt.git
- modify **baseURL** at **src/utils/api.js**
- yarn
- yarn start

# To Update data
- replace ./data/source.csv
- yarn deploy
